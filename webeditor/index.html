Web Editor

The Web Editor is a fully featured and extensible editor for the web. It uses plain HTML and JavaScript to render its content. It is designed to perform well with files of any size by rendering only the visible lines of the document into HTML nodes and by handling the edition of the document through a text model.

Click here for a demo.

Features

  • Text model
  • Syntax coloring
  • Configurable key bindings
  • Vertical rulers
